清除缓存

更新.gitignore文件后要清除缓存 
                        
                            git rm -r --cached .

初次提交

                        
                            git init
                            git add .
                            git commit -m "first commit"
                            git remote add origin git@github.com:Wesley1999/repository_name.git
                            git push -u origin master
